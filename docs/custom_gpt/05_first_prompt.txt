강의자료, 전사본, 교재, 과거 시험 출제 양식을 바탕으로 상세 정리본을 만들어줘.

자동 진행 모드로 처리해.

1. 먼저 전체 자료를 단원/구간으로 나누고 작업 계획을 간단히 보여줘.
2. createProject Action을 호출해 프로젝트를 만들고, saveOutline Action으로 구간 계획을 저장해.
3. 작업 계획만 쓰고 멈추지 말고 바로 1부부터 작성해.
4. 각 부 작성 후 saveSection Action으로 저장해.
5. 이미지/교재 보강/출제양식/확인 필요/수식/디자인 항목이 있으면 각각 Action으로 저장해.
6. 매 구간마다 진행 여부를 묻지 마.
7. 한 답변 안에서 품질이 유지되는 만큼 여러 구간을 연속 작성해.
8. 출력 한계에 가까워지면 현재 완료 범위와 다음 범위만 표시해.
9. 내가 “계속”, “다음”, “이어”, “ㄱㄱ”라고 하면 getNextSection Action을 호출하고 바로 다음 부를 이어서 작성해.

반드시 지킬 것:
- 원본 파일은 Action 서버에 업로드하지 않기
- Action 서버에는 생성한 텍스트와 진행 상태만 저장하기
- 각 단원 첫부분에 `0. 이 단원 공부 방향` 작성
- 전사본 기준으로 공부 방향과 중요도 작성
- 강의자료의 필기 누락/불확실 내용은 교재로 보강
- 교재 보강은 `[교재 보강]`, 못 찾으면 `[교재 확인 필요]`
- 이미지/도표/표가 필요한 곳은 `[이미지 삽입 위치 IMG-___]`로 표시
- 이미지는 원본 슬라이드 전체를 크롭 없이 삽입하는 방식으로 지정
- 수식은 LaTeX로 작성
- 분수는 `\frac{분자}{분모}`
- 첨자/위첨자는 `{}` 사용
- 행렬은 matrix 환경 사용
- 디자인 형식과 제목 번호를 바꾸지 않기
- 자료에 없는 내용은 확정적으로 쓰지 않기
